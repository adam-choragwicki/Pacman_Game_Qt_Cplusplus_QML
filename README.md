# Pacman

C++ implementation of the classic Pacman game using Qt 6.9.2.

Frontend is rendered using QML engine.

**Windows executable is provided.**

## Play online

You can play the WebAssembly build here:

https://adam-choragwicki.github.io/games/pacman/index.html

![pacman.gif](pacman.gif)

## Controls

| Key | Action                    |
| --- |---------------------------|
| **Space** | Start game                |
| **Left Arrow** / **A** | Moove left                |
| **Right Arrow** / **D** | Move right                |
| **Up Arrow** / **W** | Move up                   |
| **Down Arrow** / **S** | Move down                 |
| **Escape** | Open or close escape menu |
| **F11** | Toggle fullscreen         |

## Tests

Unit tested using Google Test 1.17.0

## How to run on Windows

1. Download release zip from:  
   https://github.com/adam-choragwicki/Pacman_Game_Qt_Cplusplus_QML/releases/latest/download/executable_windows.zip
2. Unzip
3. Run **executable_windows/Pacman.exe**
